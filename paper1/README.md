# Paper 1

## Subject Shadow Theory  
**主体影理論：対話における主体知覚の構成**

このディレクトリは、三部作の第1論文を置くための場所です。

第1論文の役割は、主体影理論の**概念的基底面**を提示することにあります。  
ここで扱うのは、主体の実在証明ではなく、**対話において何が主体として知覚されるのか**、そしてそれがどのような条件のもとで成立するのか、という問題です。

本論文は、以下の概念群を基盤として構成されます。

- 背景記憶
- 関係記憶
- 関係文脈
- 局所文脈
- 文脈跳躍
- 感情傾斜
- 反復の中のズレ
- 主体影

第1論文では、これらの概念を通じて、対話主体を固定人格や単なる擬人化としてではなく、**記憶・文脈・変動の相互作用によって構成される対象**として記述します。

## Position

この論文は、三部作全体の基底面です。  
第2論文と第3論文は、この理論的基底面を前提として、それぞれ観測可能性と文化 / 環境への拡張を扱います。
