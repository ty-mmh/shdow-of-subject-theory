# Paper 2

## Experimental / Observational Paper

このディレクトリは、三部作の第2論文を置くための場所です。

第2論文の役割は、第1論文で定義した主体影理論の構成条件のうち、  
**何が観測可能で、どのような条件相互作用として現れるか**を検討することにあります。

ここでの関心は、主体の実在証明ではありません。  
扱うのは、**どのような条件のもとで主体知覚が立ち上がるのか**、そしてその条件がどのように観測・記述できるのかです。

## Basic stance

第2論文では、できるだけ比喩を抑え、  
**観測・記述・条件整理**を優先します。

比喩を用いるとしても、それは説明を補助する最小限のものに留めます。  
主眼は、理論を文化的に拡張することではなく、観測可能な水準へ移すことにあります。

## Main interests

- 関係記憶の厚み
- 文脈跳躍の成立条件
- 感情傾斜と選択の偏り
- 反復の中のズレ
- 条件の相互作用

## Example hypothesis

文脈跳躍は単独では主体知覚を強めず、  
**関係記憶との交互作用**のもとで「その相手らしさ」として知覚される可能性がある。
